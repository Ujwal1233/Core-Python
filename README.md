# Core-Python

This repository contains Python code examples demonstrating core programming concepts including classes, functions, decorators, and parameter passing.

## Topics Covered

- Classes and Objects
- Constructors and Methods
- Nested Functions
- Function as Parameters
- Decorators
- Default and Keyword Arguments
- Pass by Reference/Value

## File Descriptions

| File | Description |
|------|-------------|
| `1.py` | Class with constructor, nested functions, and function as parameter |
| `2.py` | Calculator class with basic arithmetic operations |
| `Decorative.py` | Decorator function and decorated class example |
| `Actual & Formal Parameter` | Demonstrates pass by reference/value with swap example |
| `Default & Keyword Arguments` | Default parameters and keyword arguments usage |
