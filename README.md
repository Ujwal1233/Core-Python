# Core-Python

This repository contains Python code examples demonstrating core programming concepts including classes, functions, decorators, parameter passing, and module imports.

## Topics Covered

- Classes and Objects
- Constructors and Methods
- Nested Functions
- Function as Parameters
- Decorators
- Default and Keyword Arguments
- Pass by Reference/Value
- Function Overloading Concepts
- Module Imports

## File Descriptions

| File | Description |
|------|-------------|
| `1.py` | Class with constructor, nested functions, function as parameter, and function redefinition |
| `2.py` | Calculator class with basic arithmetic operations |
| `Decorative.py` | Decorator function and decorated class example |
| `Even or odd.py` | Basic even/odd number check using conditional statements |
| `Even or odd using function.py` | Even/odd check using a function with return value |
| `Even number in a user defined range.py` | Display even numbers within a user-specified range |
| `Even or odd in user defined range separately.py` | Display even and odd numbers separately within a range |
| `EX1.py` | Function module (fun1) for import demonstration |
| `EX2.py` | Function module (fun2) for import demonstration |
| `new.py` | Demonstrates module importing - imports functions from EX1 and EX2 |
| `Overloading.py` | Method overloading concepts in Python |
| `String into Uppercase using Decorative.py` | Decorator example for converting strings to uppercase |
| `Actual & Formal Parameter` | Demonstrates pass by reference/value with swap example |
| `Default & Keyword Arguments` | Default parameters and keyword arguments usage |

## Getting Started

1. Clone the repository
2. Run any Python file using: `python <filename>.py`

## Requirements

- Python 3.x
